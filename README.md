# Nishchal Vyas - Portfolio Website

A modern, responsive portfolio website showcasing skills, projects, and experience as a Full Stack Developer and ML Engineer.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Smooth scrolling navigation, hover effects, and form validation
- **Project Showcase**: Dedicated sections for ML and web development projects
- **Skills Display**: Organized categorization of technical skills
- **Contact Form**: Functional contact form with validation

## Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About**: Professional summary and background
3. **Experience**: Work history with timeline layout
4. **Projects**: Categorized project showcase (ML & Web Development)
5. **Skills**: Technical skills organized by category
6. **Education**: Academic background and certifications
7. **Contact**: Contact information and form

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with Flexbox and Grid
- **Icons**: Font Awesome
- **Animations**: CSS animations and Intersection Observer API

## Customization

To personalize this portfolio with your information:

1. **Update Personal Information**:
   - Replace placeholder text in `index.html` with your actual details
   - Update contact information in the contact section
   - Add your actual project links and descriptions

2. **Add Your Projects**:
   - Replace project descriptions with your actual projects
   - Update GitHub and live demo links
   - Add project screenshots or icons

3. **Customize Skills**:
   - Modify the skills list in `script.js` or directly in HTML
   - Add or remove skill categories as needed

4. **Update Experience**:
   - Replace work experience with your actual job history
   - Update dates, company names, and job descriptions

5. **Personalize Styling**:
   - Modify colors in `styles.css` to match your brand
   - Adjust fonts, spacing, or layout as preferred
